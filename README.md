# express后台开发框架简介

## 介绍 
基于express搭建的一款小巧、高性能、功能强大的后台开发框架。  
  

## 目录规划
- |- .temp          缓存目录，存放日志，临时文件等
- |- .vscode        编辑器配置，代码片段
- |- bin            启动脚本
- |- config         配置文件，路由配置
- |- controller     控制器层
- |- lib            类库和中间件
- |- public         静态目录
- |- service        业务逻辑层
- |- view           视图模板
- |- pm2.json       PM2配置
